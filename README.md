# 📊 AI Data Analyst Dashboard

This is a Streamlit-based web application that allows users to upload CSV files and instantly analyze data with visualizations and insights.

## 🚀 Features
- Upload CSV file
- Data preview
- Search functionality
- Column-wise analysis
- Statistics for numeric data
- Missing values detection
- Data type summary
- Charts (Bar, Line, Histogram, Boxplot, Pie)
- Correlation heatmap
- AI-based insights
- Download processed data

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn

## ▶️ How to Run
```bash
streamlit run app.py
